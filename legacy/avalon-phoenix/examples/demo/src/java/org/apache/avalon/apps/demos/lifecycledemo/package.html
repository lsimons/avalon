The 'Lifecycle' Demo Application

By Paul Hammant

The lifecycle demo shows only the calling of the lifecycle methods under Phoenix control. This app has meaning when you consider the lifecycle page.

Do not forget that the lifecycle methods are optional. You cann implement all of them, as this demo does, some or none of them.
