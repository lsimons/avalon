The kernel package defines a top level Kernel implementation that serves as a host to a set of blocks. The default kernel implementation handles the setup of supporting infrastructure and deployment of declared blocks.

Component Summary
{@link org.apache.avalon.merlin.impl.DefaultKernel} A {@link org.apache.avalon.merlin.Kernel} implementation that provides support the establishment, deployment and decommissioning of a set of {@link org.apache.avalon.activation.appliance.Block } instances.
