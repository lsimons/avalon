Product Description

Avalon Merlin

Merlin is the Avalon container Reference Implementation.

The Merlin system is a container that provides comprehensive support for the management of complex component-based systems. Merlin uses a component meta-model to facilitate the automated assembly and deployment of simple and composite components.
