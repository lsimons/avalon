Installation (.zip Distribution)

You can download the binary installation of MerlinDeveloper here.

To install MerlinDeveloper through our downloadable .zip file, please proceed with the following steps:

  1. Please make sure, that you have installed a valid eclipse version. (see System Requirements).

  2. Extract the content of the .zip archive 'MerlinDeveloper_[version].zip' into directory $ECLIPSE_HOME$/plugins, where $ECLIPSE_HOME$ is the directory where Eclipse is installed.  
    In Windows, this is usually C:\Eclipse
     

  3. Start Eclipse - if Eclipse is already running, please restart Eclipse.
     
  4. After you have restarted Eclipse you can check that MerlinDeveloper was successfully installed and loaded.

     

    First open the "About Dialog".
     

     

    Select the menu item Help > About
     

     

     

     

    Please press Plug-in Details button in About dialog.
     

     

    Now you should see this list:

     

     

 

If you don’t see those MerlinDeveloper Plug-ins in the list:

  1. Check that you have installed the correct version of MerlinDeveloper for the version of Eclipse that you are using.
     
  2. Check for error messages in file:     $ECLIPSE_HOME$\workspace\.metadata\.log
     
  3. Shutdown Eclipse, delete this file, start Eclipse again, and try to use MerlinDeveloper. 
    This process will remove old error messages and only include messages about the MerlinDeveloper or Eclipse loading process in the file listed above..
