System Requirements

MerlinDeveloper is developed on an NT/2000/XP basis. Also MerlinDeveloper should run on all other operating
systems on which the Eclipse environment is available it is not tested yet.

If anybody is testing MerlinDeveloper on other platforms, please let us know whether you succeeded.

MerlinDeveloper is tested on the following Eclipse Versions:

  • Eclipse 2.0.x
  • Eclipse 2.1.x
  • Eclipse 3.0 (M5)
  • Eclipse 3.0 (M6) Build: I20040121. Because of too many bugs the official M6 distribution is not recommended!

 

Right now, MerlinDeveloper is only running with JDK1.4 or higher. 

Tests with IBM JDKs are on the way.

 

Note: Either if you start MerlinDeveloper the first time or you create your first Merlin Container project, you have to be connected to the internet! This is because of a repository service of Merlin, which automatically reloads all needed resources from a central repository over the internet.
