
                                   LogKit 
                       
  What is it?
  -----------

  It is a lightweight, fast, securely designed logging toolkit. It is designed to integrate 
  into existing applications.

  Where is it?
  ------------

  http://avalon.apache.org/logkit

  Requirements
  ------------

  -JDK1.2 or above
  -To build form CVS you must set JAVA_HOME to the jdk dir (eg:/usr/bin/jdk1.2 or 
   c:\jdk1.3)
